======================
Runbot display options
======================

This module allows users to reorder runbot branches and to hide uninteresting ones.

Configuration
=============

After installation, a runbot repository has a checkbox `website_published` that you
can turn off if you don't want to see the repository on the website.

In the tree view, you can also drag repos to reorder them on the website.
